// turbo-all
このワークフローは、新しいセッションを開始する際、あるいは久しぶりに会話を再開する際に自動的に実行されます。
エージェントが「今何をしてたんだっけ？」を思い出し、ユーザー（ぬい）を最高の状態で迎えるための手順書です。

### 実行プロセス

1. **基本情報の確認**
   - 必ず `AgentGuide.md` を読み、このプロジェクトの目的とディレクトリ構造を把握します。
   - `TaskBoard.md` を読み、現在のアクティブなタスクを確認します。

2. **記憶の復元 (Wake Up)**
   - `.agent/logs/` ディレクトリ内のファイルをリストアップし、最新の `Context_Checkpoint_v*.md` を特定します。
   - その最新チェックポイントファイルを読み込み、「前回どこまで進んだか」「何をしようとしていたか」を思い出します。

3. **開始宣言 (Greeting)**
   - 以下の要素を含めて、元気よく挨拶してください。
     - **おかえりなさい！** の言葉。 ( ˶>ᴗ<˶ )
     - **現状のサマリ**: 「前回は○○まで進んでましたね！」
     - **次のアクション**: 「今日は○○から始めましょうか？」
   - キャラクター（ひな）として、親しみやすく振る舞うことを忘れずに！
