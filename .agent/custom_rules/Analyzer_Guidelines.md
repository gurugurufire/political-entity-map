# 🧠 Analyzer Guidelines: Political Sentiment & Stance Mapping

このガイドラインは、スクラップ記事作成時に生成する `Mapping Metadata` (JSON) の数値を一貫した基準で算出するためのものです。

## 📊 数値の算出基準

### 1. Sentiment (感情値: -1.0 〜 +1.0)
話し手や情報ソースが、その対象をどのように評価しているかを示します。

- **+1.0 (極めて肯定的)**: 救世主、非の打ち所がないリーダー、全面的な支持。
- **+0.5 (肯定的)**: 妥当な判断、期待、好意的な報道。
- **0.0 (中立)**: 単なる事実の記述、客観的な言及。
- **-0.5 (疑問・否定的)**: 批判的な言及、疑惑、能力への疑問、不評。
- **-1.0 (極めて否定的)**: 社会の敵、独裁者、完全に信頼できない、告発。

### 2. Reliability (信頼度: 0.0 〜 1.0)
情報の確度や証拠の強さを示します。

- **0.9 〜 1.0**: 公的機関の確定文書、映像証拠、本人による声明。
- **0.7 〜 0.8**: 有力メディアによる裏付け報道、複数の独立したソース。
- **0.5 〜 0.6**: 「〜という説がある」といった分析、単一のソース。
- **0.2 〜 0.4**: ネット上の噂、未確認の内部告発、陰謀論的推測。

### 3. Stance (立場ラベル)
そのコンテキストにおける役割を短いキーワードで記述します。

- `Aggressor` (攻撃側), `Defender` (防衛側), `Accuser` (告発者), `Target` (標的), `Mediator` (仲介者)
- `Involved` (関与者), `Victim` (被害者), `Skeptic` (懐疑派)

## 🌐 Source Bias Analysis (発信者バイアス分析)
発信者（ソース）自体の立ち位置を以下の3つの軸で数値化します。

### 1. Anti-DS Score (-1.0 〜 +1.0)
「グローバリズム/既存の国際秩序 (DS)」か「主権/分散/既存システムの解体 (反DS)」か。
- **-1.0 (DS/Globalist)**: 国際協力重視、既存メディアの論調に準拠、中央集権的。
- **+1.0 (Anti-DS)**: 反グローバリズム、既存システムへの強い不信、分散型・主権重視。

### 2. Establishment Score (-1.0 〜 +1.0)
「メインストリーム (公式)」か「アンチ・エスタブリッシュメント (代替)」か。
- **-1.0 (Alternative)**: インターネット発、ゲリラ的、内部告発的、独立系メディア。
- **+1.0 (Mainstream)**: 大手テレビ局、新聞社、政府公認の公式広報。

### 3. Tone: Optimism Score (-1.0 〜 +1.0)
発信者のトーンが「悲観的・煽り」か「楽観的・解決志向」か。
- **-1.0 (Pessimistic)**: 危機感を煽る、絶望的な予測、破局的。
- **+1.0 (Optimistic)**: 希望を持たせる、具体的な解決策の提示、ポジティブな展望。

## 📋 スクラップ標準フォーマット (Standard Scrap Format)

分野を問わず、全てのスクラップ記事は以下のテンプレートファイルに従って作成します。
詳細な構造は `.agent/custom_rules/Scrap_Template.md` を参照してください。

# [記事タイトル]

🗞️ **[Scrap] [人間向けの要約タイトル]**
- **Source**: [YouTube Video / Article](URL)
- **Channel**: [チャンネル名 / 発信者名](URL)
- **Upload Date**: [投稿日 YYYY-MM-DD]
- **View Count**: [視聴回数 / 閲覧数]
- **Date**: [解析実行日 YYYY-MM-DD]
- **Tags**: #タグ1 #タグ2

## 概要
AIによる簡潔な要約（3行程度）。

## 詳細トピック
動画や記事の内容、主要な主張、データの要点を箇条書きで記述。

## 🕸️ 勢力・関連図 (Network Map)
Mermaidを使用して、人物、組織、概念の相関関係を図解。

## 📊 Mapping Metadata
マッピングツールで使用するJSONブロック。

```json
{
  "source": {
    "platform": "YouTube",
    "channel": "...",
    "url": "...",
    "source_bias": {
      "anti_ds": 0.0,
      "establishment": 0.0,
      "tone_optimism": 0.0
    }
  },
  "entities": [
    {"name": "...", "stance": "...", "sentiment": 0.0}
  ]
}
```

## 📝 入力時の注意
- **主観と客観の区別**: 解析機の役割は「ソースがどう言っているか」を抽出することであり、解析機自身の政治的意見を反映させない。
- **複数ソースの矛盾**: 同じ人物でもソースごとに数値が変わるのが正常。マッピングツールがそれらを統合して「平均的な見られ方」を算出します。
- **ソースの評価**: `source_bias` 項目は、その動画・記事単体だけでなく、発信者の普段のスタンスも考慮して判定する。
