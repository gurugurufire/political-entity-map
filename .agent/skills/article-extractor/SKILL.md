# 📝 Article Extractor (記事抽出スキル)

このスキルは、ウェブサイトからノイズ（広告、ナビゲーション、サイドバー）を除去し、記事の本文だけを綺麗に抽出するためのツールを提供します。

## 🚀 主な機能

1.  **本文抽出**: 記事の主要なテキストのみを抽出します。
2.  **メタデータ取得**: タイトル、著者、公開日などの情報を取得します。
3.  **クリーンな出力**: Markdown形式やプレーンテキストで、読みやすい形で保存します。

## 🛠 使い方

### 1. 記事の抽出と要約
URLを指定して、内容を読み取ります。

```bash
# 内部的に Python スクリプトを使用して抽出します
python .agent/skills/article-extractor/scripts/extract_article.py "URL" --output article.md
```

### 2. フロー
1.  **抽出**: `scripts/extract_article.py` を実行して、URLからコンテンツを取得します。
2.  **保存**: 抽出された内容を `docs/research/articles/` フォルダなどに保存します。
3.  **活用**: 要約を作成したり、知識ベース（Knowledge Base）に追加したりします。

## 🔧 推奨ツール (Requirements)

このスキルは以下のライブラリがインストールされていると、より高い精度で動作します：

-   `trafilatura`: 非常に高精度な記事抽出ライブラリ。
-   `beautifulsoup4`: フォールバック用。

## 💡 使用例

-   「この最新ニュースの要約を作って： [URL]」
-   「この記事の重要なポイントを箇条書きにして： [URL]」
-   「この記事をリサーチ資料として docs フォルダに保存して： [URL]」

## 📋 運用ガイドライン

-   抽出した記事を保存する際は、ファイル名をタイトルに基づいたわかりやすいものにしてください（例: `2026-01-30-AI-Research.md`）。
-   **出典の明記**: 抽出した内容をドキュメント化する際は、必ず **ソースURL、サイト名、公開日** を冒頭に明記してください。
-   複数の記事を比較分析する場合は、一度すべて抽出してから比較ステップに入ってください。
