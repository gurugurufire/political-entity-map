# 📺 YouTube Transcript (YouTube文字起こしスキル)

このスキルは、YouTube動画の字幕データを取得し、テキスト形式で抽出するためのツールを提供します。

## 🚀 主な機能

1.  **文字起こし取得**: 動画IDまたはURLから字幕データをテキストとして取得します。
2.  **メタデータ取得**: 動画タイトル、チャンネル名、投稿日、視聴回数、タグなどの詳細情報を取得します。
3.  **多言語対応**: 日本語、英語など、利用可能な字幕を選択して取得可能です。
4.  **タイムスタンプ**: 必要に応じて、時間情報を含めた形式での出力もサポートします。

## 🛠 使い方

### 1. 文字起こしの取得
URLを指定して、テキストを抽出します。

```bash
python .agent/skills/youtube-transcript/scripts/get_transcript.py "https://www.youtube.com/watch?v=VIDEO_ID" --output transcript.txt
```

### 2. フロー
1.  **取得**: `scripts/get_transcript.py` を実行して、動画の字幕を取得します。
2.  **要約**: 取得した長いテキストを、エージェントが要約します。
3.  **保存**: `docs/research/transcripts/` フォルダに保存して記録に残します。

## 🔧 必要ツール (Requirements)

-   `youtube-transcript-api`: YouTubeの字幕を取得するための主要ライブラリ。

## 💡 使用例

-   「このYouTube動画の内容を3行で要約して： [URL]」
-   「この動画の5分あたりの内容を詳しく教えて： [URL]」
-   「リサーチのために、この動画の全スクリプトを保存して： [URL]」

## 📋 運用ガイドライン

-   保存する際は、動画タイトルやチャンネル名をファイル名に含めると管理しやすくなります。
-   **出典の明記**: 取得した内容をドキュメント化する際は、必ず **動画のURL、チャンネル名、動画タイトル** を冒頭に明記してください。
-   自動生成字幕の場合、誤字が含まれる可能性があることを念頭に置いて要約してください。
-   非常に長い動画の場合は、セクションごとに区切って処理することを推奨します。
